UijeongBoo Design Framework

모듈화된 미니멀 다크 테마 디자인 시스템. 라임 그린 포인트 컬러와 투명 버튼 시스템을 특징으로 합니다.

디자인 규칙 강제화 활성

이 페이지는 UijeongBoo 디자인 규칙을 자동으로 감시하고 위반사항을 수정합니다.

핵심 기능

모듈화 구조

CSS, JavaScript, HTML을 기능별로 분리하여 유지보수성과 재사용성을 극대화했습니다.

디자인 규칙 강제화

라임 그린 외 색상 사용 금지, 투명 버튼 강제 등 핵심 디자인 원칙을 자동으로 강제 적용합니다.

반응형 디자인

모바일부터 데스크톱까지 완벽하게 대응하는 반응형 컴포넌트와 유틸리티 클래스를 제공합니다.

컴포넌트 시스템

사이드바, 헤더, 버튼 등 재사용 가능한 컴포넌트들을 독립적으로 사용할 수 있습니다.

파일 구조

CSS 모듈

  • css/core/ - 핵심 스타일
  • css/components/ - 컴포넌트
  • css/utilities/ - 유틸리티

JavaScript 모듈

  • js/core/ - 프레임워크 코어
  • js/components/ - 컴포넌트 로직
  • js/utils/ - 유틸리티 함수

디자인 규칙

허용되는 것

  • • 라임 그린 (#32CD32) 포인트 컬러
  • • 그레이스케일 색상 팔레트
  • • 투명 버튼 배경
  • • 플랫 디자인 (그림자 없음)

금지되는 것

  • • 라임 외 모든 색상
  • • 버튼 배경색
  • • Box-shadow 속성
  • • 이모지 사용
