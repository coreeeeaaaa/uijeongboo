사용 가이드라인

각 컴포넌트의 활용 방법과 커스터마이징 가이드

Modal Components

  • 체크박스 ID는 고유해야 합니다
  • 배경 클릭 또는 X 버튼으로 닫기 가능
  • 애니메이션은 CSS transform으로 구현
  • 모바일에서도 반응형으로 작동

Tooltip Components

  • data-tooltip 속성에 내용 입력
  • 호버 시 자동으로 표시됩니다
  • 위치는 CSS로 조정 가능
  • 말풍선 화살표 포함

Dropdown Components

  • 체크박스로 열기/닫기 제어
  • 배경 클릭으로 자동 닫힘
  • 키보드 접근성 고려 필요
  • 메뉴 항목은 자유롭게 추가 가능

Image Overlay Effects

  • zoom-effect, fade-effect, slide-effect 클래스 사용
  • 이미지 대신 배경색으로 대체 가능
  • 오버레이 내용은 HTML로 자유롭게 구성
  • 모든 효과는 CSS transition 활용
