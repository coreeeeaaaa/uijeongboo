2. Accordion Component

CSS-only accordion with smooth expand/collapse animations

UijeongBoo is a strict design system using only lime green (#32CD32) and grayscale colors. All UI elements use transparent backgrounds and flat design as default, never using shadows or glow effects.

  • • All colors except lime green are forbidden
  • • Transparent button backgrounds enforced
  • • Box-shadow property forbidden
  • • Emoji usage forbidden

All interactive elements are implemented with pure CSS without JavaScript. Using checkbox :checked states, radio button state changes, and :hover effects to create complex UI behaviors.

  • • :checked pseudo-class utilization
  • • CSS sibling selectors (+ ~) usage
  • • Transform and transition combinations
  • • Keyframes animations

Provides a total of 70 CSS-only icon animations: 10 hamburger menus, 10 search icons, 10 login icons, 10 payment icons, 10 save icons, 10 download icons, and 10 upload icons.

  • • 70 interactive icons
  • • Pure CSS animations
  • • Lightweight browser performance
  • • Mobile responsive support

All components apply responsive design that perfectly adapts from desktop to mobile. Using CSS Grid and Flexbox to provide optimal user experience across various screen sizes.

  • • CSS Grid based layout
  • • Mobile touch friendly
  • • Fluid typography scaling
  • • Adaptive component sizing
